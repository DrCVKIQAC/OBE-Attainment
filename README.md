## 📊 Overview

This is a **complete OBE (Outcome-Based Education) automation tool** designed for faculty to:

✔ Map **CO → PO / PSO**  
✔ Upload **CIA Marks (PDF / Manual)**  
✔ Generate **Attainment Levels Automatically**  
✔ Export **Excel + PDF Reports (NAAC Ready)**  
✔ Create **Action Plans Automatically**

💡 Built specifically for academic institutions aiming for **NAAC accreditation, IQAC documentation, and audit readiness**.

## 🌟 Key Features

### 🔗 CO-PO Mapping
- Map Course Outcomes to Programme Outcomes
- Supports:
  - Strong (3)
  - Moderate (2)
  - Weak (1)
  - No Mapping (0)

---

### 📄 CIA Data Handling
- Upload **PDF CIA Reports**
- Auto-extract:
  - Roll Numbers
  - Marks
- Or enter data manually

---

### 📊 Attainment Calculation
- CO-wise attainment %
- Threshold-based evaluation
- Automatic Level classification:
  - ✅ Level 3 (Achieved)
  - ⚠ Level 2 (Partial)
  - ❌ Level 1 (Low)

---

### 📈 Smart Analysis
- Student-wise performance
- CO attainment summary
- Visual classification of results

---

### 📌 Auto Action Plan (IQAC Ready)
- Strengths identified
- Weak areas highlighted
- Recommendations generated automatically

---

### 📥 Export Reports
Download in one click:

📊 Excel Report includes:
- CO-PO Mapping
- Marks Sheet
- CO Attainment
- Final Summary
- Action Plan

🖨️ PDF Report:
- Print-ready
- Signature blocks:
  - Faculty
  - HOD
  - IQAC
  - Principal
